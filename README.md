Vulntron using python 3.7<br/>


## Installing

Requirements: <br/>

<li> Requirements </li>

```bash
pip install -r requirements.txt
```

Commands:

```bash
git clone https://github.com/rout369/vulntron.git
chmod 755 -R vulntron
cd vulntron
python3 vulntron.py  
```
## Usage
Basic usage:

```bash
python3 vulntron.py
```
For the Xss_Scanner:
```bash
python3 xssxplore.py --help
python3 xssxplore.py -u http://testphp.vulnweb.com
```


## Advanced usage
For the Xss_Scanner:
```bash
python3 xssxplore.py --help
```
